Contrato de Operação de Dados

Este contrato estabelece os termos para a coleta, armazenamento, uso e proteção dos dados pessoais dos usuários do site da sala de leitura. Ao utilizar este site, você concorda com os termos descritos abaixo.

1. Dados Coletados

O site coleta os seguintes dados pessoais para fins de cadastro e operação do sistema de empréstimos de livros:

  • Nome completo;
  • Telefone;
  • Email;
  • Senha de acesso.

2. Finalidade da Coleta

Os dados coletados serão utilizados para:

  • Gerenciar o cadastro de usuários na sala de leitura;
  • Controlar os empréstimos e devoluções de livros;
  • Enviar notificações sobre reservas, prazos e comunicados relevantes;
  • Garantir a segurança e autenticidade do acesso ao sistema.

3. Armazenamento e Proteção dos Dados

Os dados serão armazenados em um ambiente seguro e acessíveis apenas por pessoal autorizado. Medidas técnicas e organizacionais serão adotadas para proteger as informações contra acessos não autorizados, alterações, divulgações ou destruições.

4. Compartilhamento de Dados

Os dados pessoais dos usuários não serão compartilhados com terceiros, exceto quando exigido por lei ou com o consentimento explícito do usuário.

5. Direitos do Usuário

Os usuários têm os seguintes direitos em relação aos seus dados pessoais:

  • Solicitar a confirmação de tratamento dos dados;
  • Acessar e corrigir informações pessoais;
  • Solicitar a exclusão de seus dados, salvo quando necessário para cumprimento de obrigações legais;
  • Revogar o consentimento para tratamento de dados.

6. Prazo de Retenção

Os dados serão mantidos pelo tempo necessário para atender às finalidades descritas neste contrato ou conforme exigido por lei.

7. Alterações no Contrato

Este contrato poderá ser atualizado periodicamente. Recomendamos que os usuários revisem esta página regularmente para acompanhar as mudanças.

8. Contato

Em caso de dúvidas ou solicitações relacionadas a este contrato, entre em contato pelo email contato@vercomagestao.com.
